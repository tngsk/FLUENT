## 2024-05-23 - Form Accessibility & Focus States
**Learning:** Adding explicit `:focus-visible` outlines and linking `<label>` elements with `htmlFor` significantly improves both keyboard navigation and screen reader compatibility in dynamically generated forms. Using transitions on hover states adds subtle visual polish that makes the UI feel more responsive.
**Action:** Always ensure custom buttons have `:focus-visible` and hover states, and dynamically generated form fields have properly associated labels.
